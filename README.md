Este es un proyecto Full-Stack desarrollado para la gestión de cultivos domésticos. 

## Tecnologías utilizadas
* **Frontend:** HTML5, JavaScript Vanilla, Tailwind CSS.
* **Backend:** Node.js, Express.
* **Base de datos:** MongoDB, Mongoose.

## Cómo ejecutar el proyecto localmente
1. Descarga o clona este repositorio.
2. Abre la carpeta `mini-huertos-api` en tu terminal y ejecuta `npm install` para instalar las dependencias (se creará la carpeta node_modules).
3. Inicia el servidor con `npx nodemon server.js`.
4. Abre el archivo `index.html` de la carpeta `frontend-huertos` en tu navegador para ver la interfaz.****
